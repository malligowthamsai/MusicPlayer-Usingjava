# 🎵 Java Music Player (Local Files)

A simple desktop music player built using Java Swing that plays local MP3 files.

---

## 🚀 Features

- 🎶 Play MP3 files from your system  
- 📂 Load songs from a folder  
- 📜 Display playlist  
- ⏯ Play selected song  
- ⏭ Next / Previous controls  
- 🖥 Simple and clean GUI  

---

## 🧠 Concepts Used

- Java Swing (GUI)
- File Handling (File, FileInputStream)
- Multithreading (for music playback)
- External Library Integration (JLayer)

---

## 📦 Requirements

- Java (JDK 8 or above)
- JLayer Library (`jl1.0.1.jar`)
